Implementation of projective texture mapping example using basic opengl funcions. The original code was found in "ftp://ftp.sgi.com/sgi/opengl/contrib/mjk/tips/projtex/index.html".

To run with the image texture, include the file path as argument.

Also, see 'http://www.informit.com/articles/article.aspx?p=770639&seqNum=4' for more information about the projective texture mapping implementation.
